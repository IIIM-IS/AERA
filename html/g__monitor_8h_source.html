254 // Case A: target==actual goal and target!=f_imdl: simulations have been produced for all sub-goals.
255 // Wait until the STHZ, in the meantime:
256 // if input==goal target, assert success and abort: invalidate goal (this will invalidate the related simulations).
257 // if input==|goal target, assert failure and abort: the super-goal will probably fail and so on, until some drives fail, which will trigger their re-injection.
258 // if input==pred goal target, do nothing.
259 // if input==pred |goal target, do nothing.
260 // if input==pred success/failure of any other goal and sim->super-goal==goal, store the simulation for decision at STHZ.
261 // if input==pred goal target and sim->super-goal==goal, store the simulation for decision at STHZ.
262 // if input==pred |goal target and sim->super-goal==goal, store the simulation for decision at STHZ.
263 // if input==pred goal target and sim->super-goal!=goal, predict success for the goal.
264 // if input==pred |goal target and sim->super-goal!=goal, predict failure for the goal.
265 // At STHZ, choose the best simulations if any, and commit to their sub-goals; kill the predictions for the discarded simulations.
266 //
267 // Case B: target==f_imdl; this means we need to fullfill some requirements: simulations have been produced for all the sub-goals of f_imdl.
268 // Wait until the STHZ, in the meantime:
269 // if input==pred success/failure of any goal and sim->super-goal==goal, store the simulation for decision at STHZ.
270 // At STHZ, choose the best simulations if any, and commit to their sub-goals; kill the predictions for the discarded simulations.
271 //
272 // Case C: target==simulated goal and target!=f_imdl: simulations have been produced for all sub-goals.
273 // Wait until the STHZ, in the meantime:
274 // if input==goal target, predict success for the goal and abort: invalidate goal.
275 // if input==|goal target, predict failure for the goal and abort: invalidate goal.
276 // if input==pred success/failure of any other goal and sim->super-goal==goal, store the simulation for decision at STHZ.
277 // if input==pred goal target and sim->super-goal==goal, store the simulation for decision at STHZ.
278 // if input==pred |goal target and sim->super-goal==goal, store the simulation for decision at STHZ.
279 // if input==pred goal target and sim->super-goal!=goal, predict success for the goal.
280 // if input==pred |goal target and sim->super-goal!=goal, predict failure for the goal.
281 //
282 // Case D: target==simulated f_imdl; this means we need to fullfill some requirements: simulations have been produced for all the sub-goals of f_imdl.
283 // Wait until the STHZ, in the meantime:
284 // if input==pred success/failure of any goal and sim->super-goal==goal, store the simulation for decision at STHZ.
285 // At STHZ, choose the best simulations if any, and commit to their sub-goals; kill the predictions for the discarded simulations.
286 }
